## Описание

Шаблонный проект для создания сервера на базе NestJS.

Включает в себя использование следующих инструментов и технологий:
* Docker
    * Postgres
    * Redis
* NestJS
* TypeORM
* ESLint
* Prettier
* @proscom/typeorm-uml
* bull
* [graphql-faker](https://github.com/Proscom/graphql-faker/tree/overrides)
